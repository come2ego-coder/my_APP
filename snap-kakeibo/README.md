# パシャ家計簿

レシートの写真を撮る(か、ライブラリから選ぶ)だけで、AIが店名・金額・カテゴリを
読み取って家計簿に記録してくれるアプリです。面倒な手入力はほぼ不要で、確認して
「保存する」を押すだけなので、ズボラでも続けやすい設計になっています。

- ホーム画面には今月の合計・カテゴリ別の内訳(横棒グラフ)・日別の記録一覧が
  並び、月ごとに左右の矢印で切り替えられます。
- 右下の📷ボタンで撮影/画像選択 → サーバー側で Gemini API に画像を送って
  店名・日付・金額・カテゴリ・メモを推測 → 内容を確認して保存、という流れです。
  読み取りに失敗しても手入力でそのまま保存できます。1枚のレシートに食料品と
  日用品などカテゴリが混在している場合、AIが自動でカテゴリごとに金額を
  分けて提案します(内容は保存前に確認・調整できます)。
- 読み取りは数秒かかりますが、その間も📷ボタンで次のレシートを続けて撮れます。
  撮った写真は「📥 確認待ち」に並び、読み取りが終わったものから好きな順番で
  タップして内容を確認・保存できます。
- 「✏️ 手入力で追加」から、写真なしで直接入力することもできます。入力画面の
  上部で「支出」「収入」を切り替えられるので、給与や臨時収入もここから記録できます。
- 「定期・予定」に家賃やサブスク、給与などを登録しておくと、毎月「追加する」を
  タップするだけで(金額はその場で調整可能)その月の記録として反映されます。
  記録済みの項目は「✓ 記録済み」と表示され、タップすると内容を編集できます。
- 1枚のレシートに食料品と日用品などカテゴリが混在している場合は、入力画面の
  「✂️ 別のカテゴリに分ける」から金額の一部を別カテゴリに分けて、2件以上の
  記録として保存できます。
- 記録はブラウザの `localStorage` に保存されます。ログインしない場合、この端末・
  このブラウザだけのデータになります。
- 画面上部の「アカウントを作る」からID・パスワードを登録すると、以後の記録は
  サーバー(Upstash Redis)にも保存されるようになり、別の端末で同じIDにログイン
  すれば同じ記録が見られます。ログインは任意で、しなくてもアプリは通常通り
  使えます。
- レシート画像は読み返せる程度の画質を保った状態で保存され(記録の編集画面で
  タップすると拡大表示できます)、AI解析用に送る画像は別途その場で縮小してから
  送信しています。この画像はサーバーには同期されず、端末内(localStorage)だけ
  に保存されます。

## セットアップ

```bash
npm install
cp .env.example .env.local
# .env.local に GEMINI_API_KEY を設定 (https://aistudio.google.com/apikey で無料取得)
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開くと使えます。
`GEMINI_API_KEY` が未設定でもアプリ自体は動きますが、写真からの自動読み取りは
できず、常に手入力での保存になります。

## 環境変数

| 変数名 | 必須 | 説明 |
| --- | --- | --- |
| `GEMINI_API_KEY` | - | Google Gemini APIキー。サーバーサイドのみで使用。未設定でも動くが自動読み取りは無効になる。 |
| `GEMINI_MODEL` | - | 読み取りに使うモデルID。未設定時は `gemini-flash-latest`(画像入力に対応したGoogle管理の最新Flashモデル)。 |
| `KV_REST_API_URL` / `KV_REST_API_TOKEN` | - | Upstash Redis連携時にVercelが自動設定。ログイン機能(アカウント)を使う場合に必要。未設定でもアプリは端末保存のみで動く。 |
| `AUTH_SECRET` | - | ログインのセッションCookieに署名するためのランダムな文字列。ログイン機能を使う場合に必要。 |

## Vercelへのデプロイ

1. このリポジトリ(このディレクトリ)をVercelにインポートする。
   - Root Directory に `snap-kakeibo` を指定する。
2. Vercelのプロジェクト設定 → Environment Variables に `GEMINI_API_KEY` を追加する。
3. (任意)ログイン機能を使う場合:
   - プロジェクトの Storage タブから Upstash Redis を追加する(`KV_REST_API_URL` /
     `KV_REST_API_TOKEN` が自動で設定される)。
   - Environment Variables に `AUTH_SECRET` を追加する(適当なランダム文字列でよい)。
4. デプロイ後に発行されるURLをスマホのホーム画面に追加しておくと、アプリのように
   ワンタップで開けて便利です。

## 使ってみたときのイメージ

1. 買い物のあと、📷ボタンをタップしてレシートを撮る。
2. 数秒待つと店名・金額・カテゴリが自動で入力される。
3. 内容が合っていれば「保存する」を押すだけ。間違っていればその場でサッと直せる。
4. ホーム画面で今月使った額とカテゴリ別の内訳がひと目でわかる。
