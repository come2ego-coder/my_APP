# パシャ経費

個人事業主向けの、売上・仕入・経費を記録する帳簿アプリです。領収書の写真を撮る
(か、ライブラリから選ぶ)だけでAIが支払先・金額・科目を読み取ってくれるので、
日々の記帳はほぼ確認して保存するだけ。年末には確定申告の収支内訳書にそのまま
転記できる形で年間集計を書き出せます。

- ホーム画面には今月の利益(差引金額)・売上・仕入・経費の内訳、経費の科目別
  内訳(横棒グラフ)、日別の記録一覧が並び、月ごとに左右の矢印で切り替えられます。
- 記録は「💰売上」「🛒仕入」「🧾経費」の3種類。経費だけ、確定申告の収支内訳書
  (白色申告用)にある科目(旅費交通費・通信費・接待交際費・消耗品費など18科目)
  から選べるようになっていて、入力した内容がそのまま申告書の科目に対応します。
- 右下の📷ボタンで撮影/画像選択 → サーバー側で Gemini API に画像を送って
  支払先・日付・金額・科目・メモ(摘要)を推測 → 内容を確認して保存、という
  流れです。読み取りに失敗しても手入力でそのまま保存できます。
- 「✏️ 手入力で追加」から、写真なしで直接入力することもできます。
- 「よく使う項目」に定期券や毎月の通信費、定額の売上などを登録しておくと、
  毎月「追加する」をタップするだけで(金額はその場で調整可能)その月の記録
  として反映されます。
- 「🧾 確定申告用の年間集計」から、選んだ年の売上・仕入・経費科目別の合計と
  差引金額(所得金額)を確認できます。「📤 確定申告用の科目別CSVを書き出す」
  で収支内訳書の科目順にCSVを書き出せるので、そのまま転記や税理士への共有に
  使えます。「📤 年間の取引明細CSVを書き出す」で年間の全取引も書き出せます。
- 今月の合計カード内の「📤 この月をCSVで書き出す」から、表示中の月の取引明細
  だけを書き出すこともできます。
- 1枚の領収書に通信費と消耗品費などの科目が混在している場合は、入力画面の
  「✂️ 別の科目に分ける」から金額の一部を別科目に分けて、2件以上の記録として
  保存できます(経費のみ)。
- 記録はすべてブラウザの `localStorage` に保存されます(サーバーには保存されません)。
  端末やブラウザを変えると記録は引き継がれないので注意してください。
- 領収書画像は圧縮した画像を保存し、AI解析用に送る画像もその場で縮小してから
  送信しています。記録の写真はタップすると拡大表示できます。

このアプリが対応しているのは白色申告・収支内訳書レベルの集計までです。
65万円/55万円控除の青色申告(複式簿記・仕訳帳・貸借対照表が必要)には対応
していません。棚卸(在庫)も扱っておらず、仕入高はその月の仕入額をそのまま
合計します。

## セットアップ

```bash
npm install
cp .env.example .env.local
# .env.local に GEMINI_API_KEY を設定 (https://aistudio.google.com/apikey で無料取得)
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開くと使えます。
`GEMINI_API_KEY` が未設定でもアプリ自体は動きますが、写真からの自動読み取りは
できず、常に手入力での保存になります。

## 環境変数

| 変数名 | 必須 | 説明 |
| --- | --- | --- |
| `GEMINI_API_KEY` | - | Google Gemini APIキー。サーバーサイドのみで使用。未設定でも動くが自動読み取りは無効になる。 |
| `GEMINI_MODEL` | - | 読み取りに使うモデルID。未設定時は `gemini-flash-latest`(画像入力に対応したGoogle管理の最新Flashモデル)。 |

## Vercelへのデプロイ

1. このリポジトリ(このディレクトリ)をVercelにインポートする。
   - Root Directory に `snap-keihi` を指定する。
2. Vercelのプロジェクト設定 → Environment Variables に `GEMINI_API_KEY` を追加する。
3. デプロイ後に発行されるURLをスマホのホーム画面に追加しておくと、アプリのように
   ワンタップで開けて便利です。

## 使ってみたときのイメージ

1. 請求書を発行したら「💰売上」で、仕入れたら「🛒仕入」で記録する。
2. 打ち合わせや出張のあと、📷ボタンをタップして領収書を撮る(デフォルトは
   「🧾経費」)。数秒待つと支払先・金額・科目が自動で入力される。
3. 月末には「📤 この月をCSVで書き出す」で明細を、年末には「🧾 確定申告用の
   年間集計」から科目別CSVを書き出して、確定申告書の作成に使う。
