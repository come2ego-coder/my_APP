# YouTube文字起こしツール

YouTube動画のURLを入力すると、その動画に付いている字幕(手動字幕 / 自動生成字幕)を
取得して文字起こしを表示するツールです。APIキーや外部の文字起こしサービスは使わず、
YouTube側が既に持っている字幕データを取得・整形して表示します。

- 対応URL: 通常の `watch?v=`、短縮URL `youtu.be/`、`shorts/`、`embed/`、動画IDの直接入力
- 字幕が複数言語ある場合は言語を切り替えて再取得可能(日本語 → 英語 → その他の順で自動選択)
- タイムスタンプ付き/なしの表示切り替え、クリップボードへのコピー、`.txt` ダウンロードに対応
- Next.js (App Router) + Tailwind CSS で構築。字幕取得はサーバー側 (`app/api/transcript/route.ts`) で行うため、ブラウザからYouTubeへ直接アクセスする際のCORS制限を回避しています。

## 仕組み

1. 入力されたURLから動画IDを抽出
2. サーバー側でYouTubeの動画ページ (`watch?v=<id>`) を取得し、埋め込まれている
   `ytInitialPlayerResponse` から字幕トラック一覧 (`captionTracks`) を取得
3. 選択した字幕トラックのURLに `fmt=json3` を付けて取得し、タイムスタンプ付きテキストに整形
4. 動画に字幕が一切無い場合(手動・自動生成いずれも無い)はエラーを返す

※ この方式は動画側に何らかの字幕(自動生成でも可)が存在することが前提です。字幕が
全く無い動画は音声認識を行っていないため文字起こしできません。

## セットアップ

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開くと使えます。

## Vercelへのデプロイ

1. このリポジトリ (このディレクトリ) をVercelにインポートする。
   - Root Directory に `youtube-transcript-app` を指定する。
2. 環境変数は不要。デプロイするだけで使える。

Root Directoryの設定をVercel側で変更した場合は、このリポジトリに何かpushすると
自動で再デプロイされます。

## 制限事項

- YouTube側の非公開ページ構造に依存しているため、YouTubeの仕様変更で動作しなくなる
  可能性があります。
- 年齢制限・非公開・限定公開など、通常のページ取得でアクセスできない動画は取得できません。
- 大量のリクエストを連続で送るとYouTube側にブロックされる可能性があるため、個人利用の
  範囲での使用を想定しています。
